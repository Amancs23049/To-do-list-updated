Premium To-Do List
A sleek, premium-looking to-do list application built with pure HTML, CSS, and JavaScript. It features a modern gradient design with glassmorphism effects, task filtering, completion toggling, and a remaining task counter. Perfect for managing daily tasks with a touch of elegance.

Features

Add Tasks: Easily add new tasks via the input field and "Add Task" button (or press Enter).
Toggle Completion: Check/uncheck tasks to mark them as completed, with strike-through styling.
Delete Tasks: Remove individual tasks with a delete button.
Filter Tasks: View all tasks, only active ones, or only completed ones using filter buttons.
Task Counter: Displays the number of remaining (active) tasks.
Responsive Design: Works on desktop and mobile devices.
Animations and Effects: Smooth slide-in animations for new tasks, hover effects, and pulsing background gradients for a premium feel.
Empty State: Shows a friendly message when no tasks are available under the current filter.

Technologies Used

HTML5: For structure.
CSS3: For styling, including gradients, animations, and glassmorphism (backdrop-filter).
JavaScript (Vanilla): For dynamic functionality like adding/deleting tasks, filtering, and DOM manipulation.
No external libraries or frameworks required.

Usage

Type a task in the input field and click "Add Task" or press Enter.
Click the checkbox to toggle task completion.
Click "Delete" to remove a task.
Use the filter buttons (All, Active, Completed) to switch views.
The task counter at the bottom updates in real-time.
